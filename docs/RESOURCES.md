# Learning Resources

Curated list of books, courses, platforms, and tools for DSA interview preparation.

## Books

### Beginner-Friendly
- **Cracking the Coding Interview** by Gayle Laakmann McDowell
  - Classic interview prep book
  - 189 programming questions
  - Best for: Interview strategies and common problems

- **Elements of Programming Interviews in Python** by Adnan Aziz, Tsung-Hsien Lee, Amit Prakash
  - 250+ problems with solutions
  - Python-specific
  - Best for: Comprehensive problem coverage

### Algorithms & Theory
- **Introduction to Algorithms (CLRS)** by Cormen, Leiserson, Rivest, Stein
  - Comprehensive theoretical foundation
  - Mathematical rigor
  - Best for: Deep understanding

- **Algorithm Design Manual** by Steven Skiena
  - Practical algorithm design
  - Real-world applications
  - Best for: Problem-solving approach

### Competitive Programming
- **Competitive Programming 3** by Steven Halim & Felix Halim
  - Advanced techniques
  - Contest problems
  - Best for: Competitive programmers

## Tips for Using Resources

1. **Don't try to use everything** - Pick 2-3 main resources
2. **Quality over quantity** - Deep understanding beats breadth
3. **Practice consistently** - Daily practice is key
4. **Join a community** - Learn from others
5. **Track your progress** - Use spreadsheets or apps
6. **Review regularly** - Spaced repetition works

## Recommended Minimal Setup

For interview prep, you really only need:

1. **LeetCode** (primary practice)
2. **One good book** (CTCI or EPI)
3. **YouTube** (for visual learning)
4. **Mock interview platform** (Pramp or interviewing.io)
5. **This repository** (for pattern-based learning!)

---

Remember: The best resource is the one you'll actually use consistently!
